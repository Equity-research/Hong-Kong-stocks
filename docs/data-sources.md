# 数据源、优先级与缺失处理

## 来源等级

| 等级 | 来源 | 用途 |
| --- | --- | --- |
| A | HKEX、正式招股书、发行人公告、明确核验原始文件、Futu OpenAPI 实际字段 | 官方评分与增强评分 |
| B | etnet、富途公开财务页、AAStocks、明确转述招股书的可信媒体 | 增强评分 |
| C | 社区汇总、启发式质量分、期间不明或无法核验数据 | 仅展示 |

## 字段策略

| 字段 | 首选来源 | 降级来源 | 缺失时行为 |
| --- | --- | --- | --- |
| 招股日期、价格、手数 | HKEX / Futu OpenAPI | etnet、核验 JSON | 无日期则不进入当日可申购名单 |
| 收入、利润、毛利率、现金流、负债率 | 招股书/公司公告 | 富途财务页、可信媒体 | 保留期间和币种；不同期间不混算 |
| 基石、保荐人、绿鞋 | 招股书/HKEX | etnet | 明确无则写 `0/false` |
| 孖展 | 带时间戳的券商公开页/人工 CSV | etnet、AAStocks | 保持缺失，不推测 |
| 实际超购、配发 | HKEX 配发结果 | 无 | 公布前标记尚未公布 |
| 暗盘 | 券商暗盘公开报价 | AAStocks（可访问时） | 暗盘开始前标记尚未公布 |
| 估值 | 发行市值 + 同期间利润/收入 | 可靠同行统计 | 亏损公司用 PS；无同行则不下结论 |

## 富途与 AAStocks

- Futu OpenAPI 的 `get_ipo_list` 可用于 IPO 列表、价格、手数、入场费和可认购状态。
- IPO 期间 `get_stock_basicinfo`、快照或资金接口可能为空；不保证基石、财务或孖展。
- 富途网页财务抓取属于可选外部导入，不是默认稳定流水线，必须记录财务期间。
- AAStocks 可能返回 403，因此适配器不是默认必需数据源。

## 财务口径

- 官方视图选择最新完整财年，例如 `FY2025`。
- 增强视图可选择最新季度/中期，但收入和利润必须来自相同期间、相同币种。
- 括号负数和显式负号必须保留；亏损不得因清洗字符变为正数。
- 原始来源快照与规范化日期 JSON 分开保存，来源快照不能直接覆盖规范化结果。
